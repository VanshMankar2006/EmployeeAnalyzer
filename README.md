# EmployeeAnalyzer
A core Java application designed to analyze workforce data efficiently using modern features such as streams ,multi-threading and exception handling

🏗️ Project Architecture
The system is built across five primary modules:
Employee Class: Holding essential details like ID, Name, Department, and Salary.
Employee Manager: It manages business logic for adding, retrieving, and filtering records.
Employee Processor: It handles the Multithreading logic to process large datasets in parallel.
Exception Handling Class: A module for error management, ensuring the app doesn't crash during data mismatches.
App Class: The entry point.

🚀 Key Features
Stream API Integration
Multithreaded Processing
Error Control
Scalable CRUD Logic
